Introdução ao Ajax

Ajax: JS and XML assíncronos

Possibilita a chamada de um recurso no servidor através de um código JS no cliente(navegador)

Não é uma linguagem mas um conjunto de técnicas para se ter uma melhor interação com o usuário.
É uma requidsição que não recarrega a página. O submit do form recarrega

jQuery: é uma das bibliotecas de funções JS mais populares entre os devs e da internet. Facilita o uso de alguns efeitos, principalmente o uso de requisições AJAX. Open source e pode ser usada tanto para estudo como para fins comerciais
